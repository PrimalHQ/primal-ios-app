# Primal - iOS client
Repository of the Primal Android client for Nostr.
## Requirements
- Xcode >= 14.0
- macOS Ventura

## Building
Nothing special, just open the project in Xcode and build it.
## Dependencies
Primal iOS client uses the Swift Package Manager exclusively for dependency management.
